Workflow aligned to schema

From citizen request to accountable infrastructure action

The public journey mirrors the schema: request intake, administrative review, planning, work-order conversion, execution, and closure.

01

Submit

Citizens file a complaint or improvement request with contact details, Aadhaar-linked identity, and location text.

02

Acknowledge

CRIMS issues a public reference number immediately and stores the request in the RECEIVED state.

03

Review

City administrators evaluate feasibility and move eligible requests toward planning and work-order creation.

04

Execute

Department officers and field engineers coordinate planning, assignments, and on-ground execution.

05

Close

Validation, QC review, and project accountability make the service outcome transparent and auditable.
